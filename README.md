# todo.sh

Basic todo list made with bash.

![Short Demo](./demo.gif)

## Usage

```
todo.sh                                 List todos
todo.sh -a "Read the f*cking manual"    Add a todo
todo.sh -u 1                            Toggle todo #1 state
todo.sh -s 2 3                          Switch todo #2 with todo #3
todo.sh -rm 1                           Remove todo #1
```
